# Frontend Task

## Owner
Frontend agent

## Goal
Build the fastest possible demo UI for the MVP.

## Scope
Edit only frontend UI files unless integration is required:
- `index.html`
- `styles.css`
- `script.js` only for wiring if needed

## Required UI
- page title
- short product description
- time slot dropdown
- workout style textarea
- submit button
- result section

## Output Behavior
When the user submits:
- collect input values
- call the classifier/matching function already exposed in JavaScript
- render matched users as cards or a list

## Design Priority
- clear and demo-friendly
- readable on desktop
- no unnecessary complexity

## Do Not Spend Time On
- login
- animations
- advanced validation
- backend integration if local data works

## Done When
- form is usable
- results render cleanly
- basic styling is present
